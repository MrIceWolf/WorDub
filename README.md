# WorDub

Програмний комплекс для автоматизації підготовки субтитрів до дубляжу.

Основні можливості:

- автоматичний переклад SRT-файлів;
- транскрипція аудіо через Whisper;
- вирівнювання авторського тексту;
- формування готових SRT-файлів;
- завантаження контенту через yt-dlp.
